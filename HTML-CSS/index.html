HTML e CSS - Atividade 01

1. O que é o HTML e qual é o seu papel na construção de páginas web?

HTML é uma linguagem de marcação.

Estrutura básica de um documento HTML:

        <!DOCTYPE html>
        <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>HTML/CSS</title>
        </head>
        <body>
          <header></header>
          <main></main>
          <footer></footer>
        </body>
        </html>

2. Como o HTML se relaciona com outras tecnologias da web?

HTML cria a estrutura, CSS estiliza a estrutura e JavaScript adiciona interatividade.

3. Cite três tags HTML básicas e explique o propósito de cada uma delas.

  • div - Tag utilizada para demarcar um espaço na página.
  • form - Tag utilizada para inserir um formulário na página.
  • footer - Tag utilizada para demarcar o espaço do rodapé da página.

4. Importância das tags de listagem: ul e ol

As tags ul e ol organizam e apresentam informações de forma clara.

Lista Desordenada (ul)

Uso: Quando a ordem dos itens não é relevante.

Lista Ordenada (ol)

Uso: Quando a ordem dos itens é significativa.

5. Como a tag a é utilizada para criar links em HTML?

A tag a permite a navegação entre páginas diferentes e o download de arquivos.

6. Atributos importantes da tag a

title, rel, download, ping, hreflang.

7. Diferença entre os atributos src e alt na tag img

O atributo src indica o caminho da imagem, enquanto alt é importante para acessibilidade.

8. Principais tags usadas para criar tabelas em HTML

A tag th indica uma célula de cabeçalho, enquanto td indica uma célula normal.

9. Quando usar tabelas?

Tabelas são apropriadas para dados numéricos e estruturados, mas devem ser evitadas em casos de acessibilidade e responsividade.
